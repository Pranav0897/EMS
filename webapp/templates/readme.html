Instructions for using EMS

Loading the data

  1. Provide the path to the config .mat file in the field provided
  2. Provide the starting condition V, theta and P, Q
  3. Press submit. 1 & 2 are necessary steps

Exploring the functionality

  1. Once the data has been loaded from the previous steps, various functions such as state estimation and load flow can be done
  2. Explore these options in the tab titled operations
